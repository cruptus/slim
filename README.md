# Structure d'un projet Slim 3

Auteur : Elbaz Jérémie

## Pre-requis

- Serveur Web avec le mode rewrite
- PHP7
- Composer

## Installation 

```bash
composer install
```

> ATTENTION : Le serveur web doit pointer sur le dossier public

## Fonctionnalité 

- Routing
- Controllers
- Système de rendu avec Twig
- Erreur 404
